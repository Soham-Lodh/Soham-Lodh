# 🧠 Featured Project — PremiumIQ

### AI-Powered Insurance Premium Prediction System

🔗 Live: https://insurance-premium-predictor-drab.vercel.app  

🔗 Repository: https://github.com/Soham-Lodh/Insurance-Premium-Predictor  

### Tech Stack

React • FastAPI • XGBoost • Scikit-learn • Pandas

---

## 📌 Overview

PremiumIQ is a machine learning-powered insurance premium prediction platform that estimates insurance costs using user health, lifestyle, and financial data.

The system combines a trained XGBoost regression model with a FastAPI backend and React frontend to deliver real-time predictions with engineered risk analysis.

---

## ⚙️ Engineering Highlights

- Built complete ML pipeline from preprocessing → training → deployment  
- Implemented feature engineering for compound medical-risk evaluation  
- Designed FastAPI inference service with reusable preprocessing pipeline  
- Added schema validation and API-side error handling for unreliable inputs  
- Solved deployment issues involving CORS, environment variables, and API routing  
- Structured frontend into reusable multi-step form components  
- Optimized frontend state handling and API interaction flow

## 🩺 Featured Project — Prescripto

### Medical Appointment & Doctor Management System

🔗 Live: https://prescripto-o5lf.vercel.app  

🔗 Repository: https://github.com/Soham-Lodh/Prescripto  

### Tech Stack

MongoDB • Express • React • Node.js • JWT Authentication

---

## 📌 Overview

Prescripto is a full-stack doctor appointment and healthcare management platform that supports role-based workflows for patients, doctors, and administrators.

The project focuses on scalable backend structure, authentication systems, and efficient frontend interaction patterns.

---

## ⚙️ Engineering Highlights

- Implemented secure JWT-based authentication and authorization  
- Designed separate dashboards for admin, doctor, and patient workflows  
- Organized backend into modular routes, controllers, and middleware  
- Built REST APIs for appointments, doctor management, and user handling  
- Reduced unnecessary frontend re-renders and optimized API requests  
- Integrated Cloudinary for image upload and management
